# 微博原图下载器

一个 Tampermonkey 用户脚本，在微博网页版一键下载所有可见贴文的**原图**。

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 新建脚本，将 [weibo-image-downloader.user.js](weibo-image-downloader.user.js) 内容粘贴进去
3. 保存（Ctrl+S）
4. 打开 `weibo.com`，页面右下角会出现下载面板

## 功能

- 自动识别页面中所有微博贴文图片
- **多级画质回退**：`original → large → mw2000 → mw1024 → mw690`，自动取最高可用画质
- 画质对比测试：下载前先测试各画质级别的文件大小
- 带序号的文件名，方便整理
- 过滤 UI 素材和头像，只下载贴文内容图片

## 使用方法

1. 打开微博网页版（`weibo.com`）
2. 浏览到有图片的贴文
3. 点击右下角「📥 下载原图」按钮
4. 观察画质对比结果，脚本自动选择最高画质下载
5. 图片保存到浏览器默认下载目录

## 画质级别说明

| 级别 | 说明 |
|------|------|
| `original` | 原始上传文件（最高画质） |
| `large` | 大图（约 1280px 宽） |
| `mw2000` | 限制宽度 2000px |
| `mw1024` | 限制宽度 1024px |
| `mw690` | 标准中图 690px |
